# Chuks Ubani - Senior Frontend Engineer Portfolio

![Portfolio Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

A modern, fully responsive personal portfolio website showcasing my work as a Senior Frontend Engineer specializing in fintech, AI, edtech, and Web3 platforms.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth animations and transitions
- Read More/See Less functionality with typewriter effect
- Project portfolio with filtering
- Testimonials from industry colleagues
- Contact form
- Download CV functionality
- Dark theme with accent colors
- Performance optimized

## Tech Stack

- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript
- No external dependencies

## Getting Started

### Prerequisites

- [Git](https://git-scm.com/downloads) installed

### Installation

```bash
git clone https://github.com/yourusername/your-portfolio.git
cd your-portfolio
```

Open `index.html` in your browser or deploy to Netlify/Vercel.

## Key Customizations

This portfolio is a heavily customized version of the original vCard template. Notable changes include:

- **Typewriter Animation**: Animated text effect on About section load
- **Enhanced About Section**: Read More toggle with smooth transitions
- **Testimonials**: Real client feedback integrated into About page
- **Clients/Companies Section**: Logo showcase with proper sizing constraints
- **Custom Color Scheme**: Green accent theme matching professional branding
- **Removed Features**: Blog section removed, focused on core portfolio
- **Performance Focus**: Optimized animations and responsive images

## Original Template

This project is built upon [vCard - Personal Portfolio](https://github.com/codewithsadee/vcard-personal-portfolio) by [@codewithsadee](https://twitter.com/codewithsadee_). The original template provided the foundational structure and styling approach.

### Original Features Retained

- Skill progress bars
- Project portfolio grid
